# Polymorphism 다형성

다양한 성질로 변화시킬수 있는 구조, 형태, 특성

## Static Polymorphism 정적 다형성

- Compile Time Bind
- Early binding
- Overloading 오버로딩
- Type
- Template

## Dynamic Polymorphism 동적 다형성

- Run Time Bind
- Late binding
- Overriding 오버라이딩
- Class inheritance 클래스의 상속으로 다형성을 만듬
- Abstract Class 클래스의 추상화
